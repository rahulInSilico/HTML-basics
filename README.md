HTML Basics

This repository contains my HTML basics learning work.
I created this repository to build a strong foundation in core HTML concepts
by practicing with simple, clear, and well-structured examples.

Topics Covered (Phase 1)

1. HTML Structure
   - Understanding the basic HTML boilerplate
   - Learning how a webpage is structured using html, head, and body tags

2. HTML Headings
   - Working with heading tags from h1 to h6
   - Understanding proper heading hierarchy

3. Paragraph and Preformatted Text
   - Learning the difference between p and pre tags
   - Understanding how spaces and line breaks are handled

4. Line Break and Horizontal Rule
   - Using br to create line breaks
   - Using hr to separate content sections

Tools and Technologies
- HTML5
- Visual Studio Code

Purpose of This Repository
- To practice and strengthen HTML fundamentals
- To understand core concepts through hands-on examples
- To prepare a solid base for learning CSS and JavaScript
